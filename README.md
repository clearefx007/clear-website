CLEAR After Effects Plugin
Version: 1.0.0

HOW TO INSTALL

1. Close Adobe After Effects.
2. Run:

   CLEAR-Installer-v1.0.0.exe

3. Click:

   Install CLEAR

4. Open Adobe After Effects.
5. Go to:

   Window > Extensions > CLEAR

6. Dock the CLEAR panel wherever you want.


WHAT THE INSTALLER DOES

- Installs CLEAR into the Adobe CEP extensions folder.
- Deletes older CLEAR extension folders before installing.
- Enables the required CEP setting for After Effects.
- Keeps the panel name as CLEAR.
- No license key is needed.
